# Модуль "Промо-строка" для OpenCart (Promo Bar Module)

Модуль для создания управляемой бегущей промо-строки с настройками из админки OpenCart.

## Возможности модуля

### Управление через админку
- ✅ Включение/выключение строки
- ✅ Редактирование текста (поддержка эмодзи и HTML)
- ✅ Добавление ссылки
- ✅ Сделать всю строку кликабельной
- ✅ Выбор цвета фона
- ✅ Выбор цвета текста
- ✅ Настройка скорости прокрутки
- ✅ Период показа (дата начала/окончания)
- ✅ Выбор страниц отображения:
  - Все страницы
  - Только главная
  - Категории и главная
  - Карточки товаров и главная
- ✅ Дублирование текста для бесшовного цикла

### Технические особенности
- CSS animation (без устаревшего тега `<marquee>`)
- Бесконечная плавная прокрутка без рывков
- Адаптивный дизайн для мобильных устройств
- Пауза при наведении курсора
- z-index 9999 — поверх всех элементов
- Совместимость с темой Technics

## Установка

### Способ 1: Через админку (рекомендуется)

1. **Подготовьте файлы:**
   - Скопируйте содержимое папки `upload` в корень вашего сайта
   
2. **Установите OCMod:**
   - Зайдите в админку: `Администрирование → Расширения → Установка расширений`
   - Нажмите кнопку загрузки и выберите файл: `install/xml/promobar.ocmod.xml`
   
3. **Обновите кеш модификаторов:**
   - `Администрирование → Расширения → Модификаторы`
   - Нажмите кнопку обновления (синяя кнопка справа сверху)
   
4. **Очистите кеш темы:**
   - `Администрирование → Разработка → Переводы`
   - Нажмите кнопку обновления (синяя кнопка справа сверху)
   - `Администрирование → Разработка → Кеш тем`
   - Нажмите кнопку обновления

5. **Установите модуль:**
   - `Администрирование → Расширения → Расширения`
   - Выберите тип: `Модули`
   - Найдите в списке **"Промо-строка"** (Promo Bar)
   - Нажмите зелёную кнопку установки (+)
   - Затем нажмите синюю кнопку редактирования (карандаш)

6. **Настройте модуль:**
   - Статус: **Включено**
   - Текст промо-строки: например, `🔥 Акция! Скидка 15% на все букеты! 🔥`
   - Ссылка: URL страницы акции (или оставьте пустым)
   - Вся строка кликабельна: **Да**
   - Цвет фона: `#e74c3c` (красный) или любой другой
   - Цвет текста: `#ffffff` (белый)
   - Скорость прокрутки: `30` секунд
   - Дата начала/окончания: опционально
   - Страницы отображения: **Все страницы** (или выберите нужные)
   - Дублировать текст: **Да** (для плавной прокрутки)
   
7. **Сохраните настройки**

### Способ 2: Ручная установка (если OCMod не работает)

Если автоматическая интеграция не сработала, добавьте вызов модуля вручную:

1. Откройте файл: `catalog/controller/common/header.php`
2. Найдите строку: `$data['scripts'] = $this->document->getScripts('footer');`
3. Перед ней добавьте:
```php
// Promo Bar Module
$this->load->controller('extension/module/promobar');
$data['module_promobar'] = $this->load->controller('extension/module/promobar');
```

4. Откройте файл: `catalog/view/theme/default/template/common/header.twig`
   (или `catalog/view/theme/technics/template/common/header.twig` для темы Technics)
5. Найдите тег `<body>` и сразу после него добавьте:
```twig
{{ module_promobar }}
```

## Использование

### Быстрый запуск акции

1. Зайдите в админку: `Расширения → Расширения → Модули → Промо-строка → Редактировать`
2. Измените текст на актуальный: например, `⚡ Молния! Скидка 20% на все розы до конца дня!`
3. При необходимости установите дату окончания
4. Сохраните

**Готово!** Посетители сразу увидят новое предложение.

### Примеры текстов для промо-строки

```
🔥 Скидка 15% на все букеты до воскресенья!
🚚 Бесплатная доставка при заказе от 5000₽
💐 Новые поступления! Свежие пионы уже в продаже
⏰ Только сегодня: вторая роза в подарок!
🎁 Промокод FLOWERS10 — скидка 10% на первый заказ
```

## Настройки детально

| Параметр | Описание | Рекомендация |
|----------|----------|--------------|
| Статус | Включение/выключение | Выключайте, когда акция закончилась |
| Текст | Содержимое строки | Используйте эмодзи для привлечения внимания |
| Ссылка | URL перехода | Можно ссылаться на категорию с акционными товарами |
| Кликабельна | Вся строка как ссылка | Удобно для мобильных |
| Цвет фона | HEX цвет | Контрастный к дизайну сайта |
| Цвет текста | HEX цвет | Должен хорошо читаться |
| Скорость | Время полного цикла (сек) | 20-40 сек оптимально |
| Даты | Период показа | Автоматическое скрытие по окончании |
| Страницы | Где показывать | Зависит от типа объявления |
| Дублирование | Для бесшовности | Рекомендуется включать |

## Troubleshooting

### Промо-строка не отображается

1. Проверьте, что модуль включен (Статус = Включено)
2. Проверьте даты начала/окончания
3. Проверьте выбор страниц отображения
4. Очистите кеш: `Администрирование → Разработка → Кеш тем`
5. Обновите модификаторы: `Администрирование → Расширения → Модификаторы`

### Текст обрывается или скачет

- Включите опцию "Дублировать текст для бесшовного цикла"
- Увеличьте скорость прокрутки

### Не видно на мобильном

- Проверьте контрастность цветов
- Убедитесь, что строка не перекрывается другими элементами
